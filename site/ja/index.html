なぜ任せるのか

脚本は AI に書かせられても、成片の作業が本当の難題です。

AI に脚本を書かせるのは難しくありません。難しいのはその後——配音、タイムライン整合、レンダリング、字幕焼き付け、カバー作成、横版と縦版それぞれの調整です。

Lecturecast は成片パイプライン全体をあなたの AI エージェントに丸ごと任せます。すべてあなたのマシン上で実行され、エージェントが「テキスト」ではなく「完成した動画」を実際に届けます。

機能 · すべてローカル動作

一つのテーマから、2 本の成片へ。

i.

AI による脚本作成

テーマをエージェントに渡すと、構造化された講座脚本——セクション分け、論理的、口語調——を生成します。脚本は承認ゲート：あなたが確認・修正してから制作に進みます。

ii.

ローカル配音と整合

脚本はローカルで自然な音声に変換され、自動的に文ごとに分割、タイムラインに整合されます。長段落は長さ検証＋自動リトライで、音声と映像のズレを防止。デフォルトは無料の Edge TTS、自前 MiniMax への切り替えも可能です。

iii.

2 プラットフォーム同時レンダリング

同じ脚本から、ローカルの Remotion で B 站横版（16:9）と小红书縦版（9:16）を一度に出力。それぞれにレイアウトが最適化され、後から再編集する必要はありません。

iv.

字幕 + カバー

ローカルの ffmpeg が中国語字幕を焼き付け、2 プラットフォーム用のカバーを生成。小红书のコンプライアンスチェックも自動実行——受け取ってすぐ投稿でき、編集ソフトを開く必要はありません。

一度の生成 · 2 つのバージョン

B 站と小红书、それぞれに。

同じテーマ、同じ脚本——Lecturecast は 2 つのアスペクト比を同時に出力し、それぞれに適したレイアウトと字幕で公開可能な形に仕上げます。

1920 × 1080

B 站横版

16 : 9

知識解説や長尺動画向け；Remotion のクリーンなレイアウトに字幕焼き付け済み。

1080 × 1920

小红书縦版

9 : 16

スマホ縦向きフィード向け；Remotion の縦版レイアウト＋カバー付き。

ローカル 3 点セット依存

すべてあなたのマシン上で。

クラウドのメディア worker もリモートレンダラもありません。Director を選んでも、クラウドは創作判断と署名付き Manifest のみを担当します。3 点をインストールすれば、エージェントがメディア制作をローカルでエンドツーエンド実行できます。install.sh が不足を確認します。

RENDER

Node + Remotion

コード駆動のアニメシーンで 2 アスペクト比をレンダリング。brew install node 後 npm install。

VOICE

Python + edge-tts

ローカル配音、文ごとの分割、タイムライン整合。デフォルトは無料 Edge TTS、自前 MiniMax key でグレードアップ可能。

MUX

ffmpeg（libass 付き）

中国語字幕を焼き付け、成片を合成。brew install ffmpeg で libass 入りビルドが入手できます。

よくある質問

FAQ.

有料ですか？アカウント登録は必要？
Community は永久無料で、アカウントも LectureCast API キーも不要です。任意の Director は有効な AgentMesh360 有料サブスクリプションと共通 API キーを持つアカウントが利用できます。既存の共通 credits を使い、LectureCast 専用契約は不要です。選択と Brief 作成は無料で、ProductionManifest の生成を確認した場合のみ 10 credits を消費します。
動画はクラウドにアップロードされますか？
されません。Community は LectureCast サービスへ何も送りません。Director では限定された素材要約、安定した選択肢 ID、Brief、クライアント能力だけがクラウドへ送られます。元のメディア、配音、字幕、編集、レンダリング、完成品は常にローカルです。
配音は何を使いますか？もっと良い声に変えられますか？
デフォルトは無料の Edge TTS——すぐ使えて、出片には十分。より暖かい声が欲しい場合は、自前で MiniMax key を持ち込めます——これは Lecturecast 提供ではなく、ご自身で minimaxi.com に登録するサードパーティサービスです。key は環境変数 MINIMAX_API_KEY 経由でのみ渡され、ディスクに書き込まれることはありません。
動画は完全自動生成ですか？修正できますか？
脚本生成後に必ず一旦停止し、あなたの確認と修正を待ってから配音・レンダリングに進みます——内容はあなたが掌握し、AI は制作を担当します。
何をインストールする必要がありますか？
ローカル 3 点セット：Node + Remotion（レンダリング）、Python + edge-tts（配音）、libass 入り ffmpeg（字幕焼き付け）。install.sh が不足を教えてくれます。macOS なら brew install node ffmpeg で揃います。
どのプラットフォーム向けの成片に対応していますか？
B 站横版（1920×1080）と小红书縦版（1080×1920）を同時出力。中国語字幕焼き付け済み・カバー生成済みです。
本当にオープンソースですか？
はい。Apache 2.0。コードはすべて GitHub にあります。自由にフォーク、改造、issue 登録してください。
